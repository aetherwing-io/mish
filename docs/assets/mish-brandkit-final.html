08 Usage Guidelines

Do

  • Lowercase always: mish
  • Geist Mono for all brand text
  • Gradient flows violet → blue (never reversed)
  • μ in the wordmark gets the gradient; ish stays bright/dark
  • μ| as the terminal prompt character
  • Tighten letter-spacing at display sizes (see spec table)
  • Dark backgrounds are the default context

Don't

  • Never capitalize: Mish, MISH, MiSh
  • Don't flatten the gradient to a single color for primary use
  • Don't use green for brand — it's for success states only
  • Don't rotate, skew, or add effects to the mark
  • Don't pair with decorative or serif fonts
  • Don't describe it as "a terminal emulator"
  • Don't use default letter-spacing at sizes above 32px
